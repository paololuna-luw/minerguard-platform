# Roadmap

## V1 - Software Base

- API principal.
- Dashboard inicial.
- Modelo de datos.
- MQTT operativo.
- Simulador de telemetria.
- Alertas basicas.

- Gestion completa de minas, zonas, mineros y dispositivos.
- Historial de turnos.

- Firmware ESP32.
- Integracion LoRa.
- Pruebas con nodos reales.
- Calibracion de ubicacion por zona.

## V4 - Robustez

- Autenticacion y roles.
- Auditoria.
- Modo offline local.
- Sincronizacion remota.
- Despliegue en servidor local industrial.
